Projects

Project 1: A Movie Wesite

A movie website built with HTML, CSS and Javascript, where users navigate through a user-friendly interface, and can easily find movie details, watch trailers, and read insightful reviews. It allows users stay updated on the latest releases, upcoming attractions, and exclusive behind-the-scenes content that adds an extra layer of excitement to their movie-watching experience.

Project 2: A Furniture Website UI Design

The furniture website features a modern and minimalist design, with a clean and spacious layout that allows the furniture pieces to take center stage. The color scheme revolves around neutral tones, with a primary use of soft whites and light grays to create a calming and sophisticated atmosphere. Accents of natural wood and subtle hints of a muted pastel color add warmth and a touch of elegance to the overall design.

Project 3: Customer Support Dashboard UI Design

The customer support dashboard is a comprehensive and user-friendly interface designed to empower customer support agents with the tools and information they need to efficiently handle customer queries, issues, and requests. The design focuses on clarity, ease of navigation, and real-time data visibility.
